Why Coding House For Me

There are Four major reasons


  1. I want a fully-immersive live in bootcamp. So I can dedicate all my time focus on learning code without worrying about commuting and other daily errands. My goal to get in a bootcamp is to learn as much as possible while every single minute I'm in it. Coding House allows me to maximize the use of my time.

  2. Small class size and dedicated mentors are another reasons why I think Coding House is for me. While I have been learning programming through self-study, I know the immersion of Coding House will enable me to learn much more quickly. Here is a link to my blog about my learning journey.

  3. JavaScript is the first programming language I learned and I am sticking with it. It is the only computer language that allows developers to build a full-fledged web application through one single language. Coding House's 90% JavaScript and 10% HTML curriculum is exactly what I want.

  4. It is located in the world renowned tech hub, Silicon Valley. Students will gain exposure to the local tech community and connections with the companies by attending different tech events or hackathons. Networking with other developers help increase the job placements after the bootcamp. I also would love to expand my network from New York City to Silicon Vallery.
